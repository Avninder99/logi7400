# 7411: Tripe 3-Input AND Gate

- Number of elements: 3
- Inputs per element: 3
- Outputs per element: 1
- DIP: 14-pin

## Description

Provides three AND gates with three inputs and an output each.

## Inputs and Outputs

| Label | Description            |
|:----- |:-----------------------|
| An    | First input of gate n  |
| Bn    | Second input of gate n |
| Cn    | Third input of gate n  |
| Qn    | Output of gate n       |

## Function Table

| An  | Bn  | Cn  | Qn  |
|:---:|:---:|:---:|:---:|
| L   | X   | X   | L   |
| H   | L   | X   | L   |
| H   | H   | L   | L   |
| H   | H   | H   | H   |

H: HIGH voltage level  
L: LOW voltage level  
X: don't care

## Pin Configuration

|     | Pin | Pin |     |
|:---:|:---:|:---:|:---:|
| A1  |   1 |  14 | VCC |
| B1  |   2 |  13 | C1  |
| A2  |   3 |  12 | Q1  |
| B2  |   4 |  11 | C3  |
| C2  |   5 |  10 | B3  |
| Q2  |   6 |   9 | A3  |
| GND |   7 |   8 | Q3  |
