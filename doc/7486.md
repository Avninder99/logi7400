# 7486: Quad 2-Input XOR Gate

* Number of elements: 4
* Inputs per element: 2
* Outputs per element: 1
* DIP: 14-pin

## Description

Provides four XOR gates with two inputs and an output each.

## Inputs and Outputs

| Label | Description            |
|:----- |:-----------------------|
| An    | First input of gate n  |
| Bn    | Second input of gate n |
| Qn    | Output of gate n       |

## Function Table

| An  | Bn  | Qn  |
|:---:|:---:|:---:|
| L   | L   | L   |
| L   | H   | H   |
| H   | L   | H   |
| H   | H   | L   |

H: HIGH voltage level  
L: LOW voltage level

## Pin Configuration

|     | Pin | Pin |     |
|:---:|:---:|:---:|:---:|
| A1  |   1 |  14 | VCC |
| B1  |   2 |  13 | B4  |
| Q1  |   3 |  12 | A4  |
| A2  |   4 |  11 | Q4  |
| B2  |   5 |  10 | B3  |
| Q2  |   6 |   9 | A3  |
| GND |   7 |   8 | Q3  |
