# 7442: BCD to Decimal Decoder

- Number of elements: 1
- Inputs per element: 4
- Outputs per element: 10
- Bits: 4
- DIP: 16-pin

## Description

Decodes a 4-bit BCD to ten output lines.

## Inputs and Outputs

| Label | Description               |
|:----- |:--------------------------|
| Dn    | input lines               |
| Qn    | output lines (active LOW) |
