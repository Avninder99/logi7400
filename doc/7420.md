# 7420: Dual 4-Input NAND Gate

- Number of elements: 2
- Inputs per element: 4
- Outputs per element: 1
- DIP: 14-pin

## Description

Provides two NAND gates with four inputs and an output each.

## Inputs and Outputs

| Label | Description            |
|:----- |:-----------------------|
| An    | First input of gate n  |
| Bn    | Second input of gate n |
| Cn    | Third input of gate n  |
| Dn    | Fourth input of gate n |
| Qn    | Output of gate n       |

## Function Table

| An  | Bn  | Cn  | Dn  | Qn  |
|:---:|:---:|:---:|:---:|:---:|
| L   | X   | X   | X   | H   |
| X   | L   | X   | X   | H   |
| X   | X   | L   | X   | H   |
| X   | X   | X   | L   | H   |
| H   | H   | H   | H   | L   |

H: HIGH voltage level  
L: LOW voltage level  
X: don't care

## Pin Configuration

|     | Pin | Pin |     |
|:---:|:---:|:---:|:---:|
| A1  |   1 |  14 | VCC |
| B1  |   2 |  13 | D2  |
| -   |   3 |  12 | C2  |
| C1  |   4 |  11 | -   |
| D1  |   5 |  10 | B2  |
| Q1  |   6 |   9 | A2  |
| GND |   7 |   8 | Q2  |
