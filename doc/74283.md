# 74283: 4-bit Binary Full Adder

- Number of elements: 1
- DIP: 16-pin

## Description

Provides a 4-bit binary full adder.

## Inputs and Outputs

| Label | Description                      |
|:----- |:---------------------------------|
| An    | Input of bit n of first operand  |
| Bn    | Input of bit n of second operand |
| Qn    | Output of bit n of result        |
| CIN   | Carry input                      |
| COUT  | Carry output                     |
