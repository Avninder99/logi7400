# 74x00: Quad 2-Input NAND Gate

* Number of elements: 4
* Inputs per element: 2
* Outputs per element: 1
* DIP: 14-pin

## Description

Provides four NAND gates with two inputs and an output each.

## Inputs and Outputs

| Label | Description            |
|:----- |:-----------------------|
| nA    | First input of gate n  |
| nB    | Second input of gate n |
| nY    | Output of gate n       |

## Function Table

| nA  | nB  | nY  |
|:---:|:---:|:---:|
| L   | L   | H   |
| L   | H   | H   |
| H   | L   | H   |
| H   | H   | L   |

H: HIGH voltage level
L: LOW voltage level

## Pin Configuration

|     | Pin | Pin |     |
|:---:|:---:|:---:|:---:|
| 1A  |   1 |  14 | VCC |
| 1B  |   2 |  13 | 4B  |
| 1Y  |   3 |  12 | 4A  |
| 2A  |   4 |  11 | 4Y  |
| 2B  |   5 |  10 | 3B  |
| 2Y  |   6 |   9 | 3A  |
| GND |   7 |   8 | 3Y  |
