# 74x244 - Octal buffer with noninverted three-state outputs

* Number of elements: 8
* Inputs per element: 1
* DIP: 20-pin

## Function Table

| nOE | nAn | nYn |
|:---:|:---:|:---:|
| L   | L   | L   |
| L   | H   | H   |
| H   | X   | Z   |


H: HIGH voltage level
L: LOW voltage level
X: don't care
Z: high-impedance OFF-state
