# 74x04: Hex Inverter

* Number of elements: 6
* Inputs per element: 1
* Outputs per element: 1
* DIP: 14-pin

## Description

Provides six hex inverters.

## Inputs and Outputs

| Label | Description            |
|:----- |:-----------------------|
| nA    | Input of inverter n    |
| nY    | Output of inverter n   |

## Function Table

| nA  | nY  |
|:---:|:---:|
| L   | H   |
| H   | L   |

H: HIGH voltage level
L: LOW voltage level

## Pin Configuration

|     | Pin | Pin |     |
|:---:|:---:|:---:|:---:|
| 1A  |   1 |  14 | VCC |
| 1Y  |   2 |  13 | 6A  |
| 2A  |   3 |  12 | 6Y  |
| 2Y  |   4 |  11 | 5A  |
| 3A  |   5 |  10 | 5Y  |
| 3Y  |   6 |   9 | 4A  |
| GND |   7 |   8 | 4Y  |
